## 🚀 Как использовать

### Для разработчиков
1. Файлы готовы к использованию:
   - `/templates/index.html` — главный шаблон
   - `/static/css/style.css` — стили (обновлен)
   - `/static/js/chat.js` — клиентская логика

2. Требуемые изменения в backend (FastAPI/Flask):
```python
@app.get("/chat")
async def chat_page(request: Request):
    # Получить данные текущего пользователя
    user = get_current_user(request)
    return templates.TemplateResponse("index.html", {
        "request": request,
        "user": user
    })
```

3. WebSocket endpoint должен поддерживать типы:
   - `send_message` — отправка сообщения
   - `typing` — индикатор набора
   - `friend_request` — запрос в друзья

### Тестирование
1. Запустите сервер
2. Откройте `/chat` в браузере
3. Проверьте:
   - Отображение списка чатов (mock-данные)
   - Переключение между чатами
   - Отправку сообщений (требуется WebSocket)
   - Мобильную версию (ширина < 768px)
   - Переключение темы в настройках
